Agentic SmolVLA Experiment Matrix

A paper-safe plan after the visual-prompt related-work scan: make overlay a controlled intervention, not the claimed invention.

Rejected framing: "Oracle Point Overlay is the method." This collides with VP-VLA, TraceVLA, AVP, RoboPoint, and affordance-aware VLA work.
Adopted framing: "Agentic lightweight VLA control with verifier/retry and spatial-cue ablations." This is more defensible and experimentally cleaner.
Six controlled conditions
C0

Policy-only baseline

SmolVLA receives original RGB and task instruction. No overlay, no retry, no verifier-driven replan.

Evidence tiers and claim gates
Tier S: syntheticProjection math, rendering, encoding, and tensor-format diagnostics only.
Tier A: actual RGB fallbackShows overlays render on saved simulator frames, but does not prove oracle projection.
Tier H: actual heuristicShows a practical non-privileged cue can be computed on actual simulator frames.
Tier O: true oracleRequires same-step RGB, object pose, camera metadata, overlay, episode, step, and env success.
Ten paper guardrails
G1

No overlay novelty claim

Visual prompting is crowded. Cite the closest overlaps explicitly.

G2

Benchmark success only

Final success must come from environment flags, not internal verifier predicates.

G3

Separate verifier success

Verifier success is a control signal for retry/replan, not the task result.

G4

Separate source types

Synthetic, fallback, heuristic, and true-oracle evidence must never be mixed.

G5

Same seeds and budgets

Every condition needs identical task IDs, seeds, action budgets, and policy checkpoint.

G6

Report overhead

Latency and memory overhead matter because the project is about lightweight VLA control.

G7

Use oracle as upper bound

Oracle projection defines a target for learned affordance modules, not a deployable method.

G8

Main method is modular

The method is a test-time wrapper around a compact VLA, not a retrained monolith.

G9

Failure taxonomy

Classify spatial grounding, control, horizon, verifier, and retry-exhaustion failures.

G10

Paper claim stays narrow

Claim measured recovery under controlled settings, not universal agentic robot generality.
