Related Work Map: Affordance / Visual Prompt Interfaces for Agentic SmolVLA

Milestone report for deciding whether Oracle Point Overlay is novel enough, or should be reframed as one ablation inside a broader agentic lightweight VLA control study.

Novelty risk

Overlay itself is not enough.

Recent work already uses rendered visual anchors, traces, primitives, affordance maps, and keypoints to improve robot control. A paper that claims novelty as "draw a point on the VLA image" would be exposed to strong related-work objections.

Safer thesis

Make it an agentic lightweight VLA study.

The defensible contribution is a controlled test-time wrapper around SmolVLA-class policies: verifier, retry/replan, and spatial cues as auditable interventions with strict benchmark-defined success.
